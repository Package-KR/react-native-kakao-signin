# RNKakaoSigninExpoExample

Expo config plugin example for `@package-kr/react-native-kakao-signin`.

`app.json` follows the Expo setup from the root `README.md`.
Replace `kakaoAppKey`, `ios.bundleIdentifier`, and `android.package` with the values registered in Kakao Developers before testing with your own app.

```sh
npm install
npm run ios
```
